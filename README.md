# autoSummon
make random story

```
概要。
１００程度の羅列からひとつを選び出す。
物語に必要な要素をランダムに提供。
一度に２０章まで。１章あたりは１０ページ想定。

１．職業の決定。相方の登場人物。
２．アイテムの決定。報酬や希求物の内容。
３．依頼の方向性の決定。依頼の内容。
４．雑魚魔物の決定。魔物。
５．大物の決定。魔物。

それぞれ種を提供する。
物語はパターンで進む。
冒頭から始まり後日談で終わる。

一幕、冒頭。前振り。（自由）
二幕、冒険者の酒場。依頼を受ける。（１．職業の決定。相方の登場人物。）（２．アイテムの決定。報酬や希求物の内容。）
三幕、迷宮。ガーッと行く。問題発生。すぐ戻る。（３．依頼の方向性の決定。依頼の内容。）
四幕、冒険者の酒場。問題と解決策の提示。（自由）
五幕、再び迷宮。雑魚との戦い。（４．雑魚魔物の決定。魔物。）
六幕、迷宮深部。目的へ。活劇。（５．大物の決定。魔物。）
七幕、後日談。まとめ。（自由）

これらの方向性をランダムに提供する。
なお、冒頭、解決策、後日談、は余白で自由課題である。
```
